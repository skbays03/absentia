# CLI Reference

> _Stub. To be auto-generated from `lacuna --help` once the CLI is implemented._

Will document every subcommand and flag:

- `lacuna` — open TUI
- `lacuna init` — bootstrap config + state directory
- `lacuna check` — batch mode
- `lacuna explain <gap-id>` — full provenance for a gap
- `lacuna suppress <gap-id>` — mark as known/intentional
- `lacuna rules` / `lacuna rules show <id>`
- `lacuna groups`
- `lacuna stats`
- `lacuna watch` (post-MVP)

Plus global flags: `--json`, `--quiet`, `--verbose`, `--filter`, `--since`, `--max-gaps`, `--no-color`.
